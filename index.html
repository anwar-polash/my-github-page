My First Repo

My first git repo is all about learning git and github

1st time for new Repo

  1. git init
  2. git add .
  3. git commit -m "message"
  4. git remote add origin "repo link"
  5. git push -u origin master

For next all changes of existing Repo

  1. git add .
  2. git commit -m "message"
  3. git push
